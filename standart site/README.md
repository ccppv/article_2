# Справочник по тегам и атрибутам HTML

Краткое объяснение тегов и атрибутов, которые используются в этом сайте (и встречаются в большинстве страниц). По тому же принципу, что и справочник по CSS: что это и зачем нужно.

---

## Структура документа

Любая HTML-страница начинается с объявления типа документа и корневого элемента, внутри — голова (`head`) и тело (`body`). В голове — служебная информация и подключение стилей/шрифтов, в теле — то, что видит пользователь.

---

### Корень и оболочки

- **`<!DOCTYPE html>`**
  - Не тег, а объявление: браузер понимает, что дальше идёт HTML5. Пишется одной строкой в самом начале файла.

- **`<html>`** … **`</html>`**
  - Корневой элемент страницы. Всё содержимое (кроме `<!DOCTYPE>`) должно быть внутри него. Обычно на открывающем теге ставят атрибут **`lang`** (например, `lang="ru"`), чтобы браузер и поисковики знали язык страницы.

- **`<head>`** … **`</head>`**
  - «Голова» документа: сюда не пишут то, что видно на странице. Здесь подключают стили, шрифты, указывают кодировку, заголовок вкладки и т.п.

- **`<body>`** … **`</body>`**
  - Тело страницы. Всё, что должно отображаться (текст, кнопки, картинки, блоки), размещается внутри `body`.

---

### Служебная информация в `<head>`

- **`<meta charset="UTF-8">`**
  - Задаёт кодировку страницы. `UTF-8` — стандарт, чтобы корректно отображались русские буквы и спецсимволы. Обычно ставят первой среди `<meta>`.

- **`<meta name="viewport" content="width=device-width, initial-scale=1.0">`**
  - Настройка отображения на мобильных: ширина страницы подстраивается под ширину экрана, масштаб по умолчанию 1. Без этого на телефоне верстка часто «ломается».

- **`<title>`** … **`</title>`**
  - Заголовок страницы: показывается во вкладке браузера и в результатах поиска. Должен быть один на страницу.

---

### Подключение внешних ресурсов

- **`<link>`**
  - Подключает к странице внешний ресурс (стили, шрифты, иконки). Сам по себе не отображается. Пишется без закрывающего тега.

  - **`rel="stylesheet"`** — связь со стилями. Вместе с **`href="путь/к/файлу.css"`** подключает CSS-файл.
  - **`rel="preconnect"`** + **`href="https://…"`** — заранее установить соединение с другим сайтом (например, с сервером шрифтов), чтобы страница загружалась быстрее.
  - **`crossorigin`** — атрибут для запросов на другой домен (часто нужен при загрузке шрифтов с Google Fonts и т.п.).

- **`<script src="путь/к/файлу.js">`** … **`</script>`**
  - Подключает JavaScript-файл. Атрибут **`src`** задаёт путь к файлу. Если указан `src`, код внутри тега обычно не пишут — скрипт берётся из файла. Обычно `<script>` ставят в конце `body`, чтобы сначала отрисовалась разметка, потом выполнился скрипт.

---

### Блоки и разделение контента

- **`<div>`** … **`</div>`**
  - Универсальный блочный контейнер. Сам по себе ничего не меняет во внешнем виде — нужен для группировки элементов и применения к ним стилей (или скриптов) через классы. Блок по умолчанию занимает всю доступную ширину и переносится с новой строки.

- **`class="…"`**
  - Атрибут класса. Значение — одно или несколько слов через пробел. По классу в CSS задают стили (селектор `.имя-класса`), в JS находят элементы (`querySelector('.имя-класса')`). Один класс можно использовать у многих элементов.

---

### Заголовки и текст

- **`<h1>`** … **`</h1>`**
  - Заголовок первого уровня (главный заголовок страницы). По смыслу должен быть один на страницу. По умолчанию крупный и жирный. Дальше по иерархии идут `<h2>`, `<h3>` и т.д.

---

### Кнопки и формы

- **`<button>`** … **`</button>`**
  - Кнопка. Текст между тегами — то, что видно на кнопке. По клику можно повесить обработчик в JS. Отличия от ссылки: кнопка не ведёт на другой адрес по умолчанию, используется для действий на странице (отправить форму, открыть окно и т.п.).

---

## Атрибуты в двух словах

| Атрибут   | Где используется | Зачем |
|-----------|------------------|--------|
| **`lang`** | `<html>` | Язык страницы (`ru`, `en` и т.д.). |
| **`charset`** | `<meta>` | Кодировка документа (`UTF-8`). |
| **`name`** + **`content`** | `<meta>` | Мета-информация: описание, ключевые слова, viewport и т.п. |
| **`href`** | `<link>`, `<a>` | Адрес ресурса или ссылки. |
| **`rel`** | `<link>` | Тип связи: `stylesheet`, `preconnect`, `icon` и др. |
| **`src`** | `<script>`, `<img>` | Путь к подключаемому файлу (скрипт, картинка). |
| **`class`** | Почти любой тег | Имя класса для стилей и скриптов. |
