## Базовые свойства CSS

Этот файл — как мини-справочник по **основным** свойствам CSS. Здесь не «всё, что есть в CSS» (этого сотни штук), а то, с чем ученики чаще всего сталкиваются в вёрстке.

---

### Блочная модель (размер и отступы)

- **`width`** / **`height`**
  - Ширина и высота элемента.
  - Можно задавать в пикселях (`px`), процентах (`%`), относительных единицах (`em`, `rem`, `vw`, `vh`) и др.
- **`min-width` / `max-width`**, **`min-height` / `max-height`**
  - Минимальные и максимальные размеры. Ограничивают `width`/`height`, чтобы блок не становился слишком маленьким или большим.
- **`padding`**
  - Внутренний отступ от границы элемента до его содержимого.
  - Записывается:
    - `padding: 10px;` — один отступ со всех сторон.
    - `padding: 10px 20px;` — вертикаль/горизонталь.
    - `padding: 10px 20px 30px 40px;` — сверху / справа / снизу / слева.
- **`margin`**
  - Внешний отступ между элементом и соседями.
  - Аналогичная запись как у `padding`.
- **`border`**
  - Граница вокруг элемента.
  - Краткая запись: `border: 1px solid #000;` (толщина / стиль / цвет).
  - Часто используются `border-top`, `border-right`, `border-bottom`, `border-left`.
- **`box-sizing`**
  - Определяет, как считается ширина/высота:
  - `content-box` — по умолчанию, размер **без** padding и border.
  - `border-box` — размер **включает** padding и border (чаще удобен в реальных проектах).

### Отображение и позиционирование блоков

- **`display`**
  - Как элемент ведёт себя в потоке:
  - `block` — блочный (занимает всю ширину, переносится с новой строки).
  - `inline` — строчный (в строке текста, нельзя задать width/height).
  - `inline-block` — как строчный, но можно задавать размеры.
  - `flex` — гибкий контейнер (Flexbox).
  - `grid` — контейнер сетки (CSS Grid).
  - `none` — элемент не отображается и не занимает место.
- **`position`**
  - Способ позиционирования:
  - `static` — по умолчанию, без особого позиционирования.
  - `relative` — относительно своего обычного положения (`top`, `left` и т.д. двигают его).
  - `absolute` — вырывается из потока и позиционируется относительно ближайшего предка с `position: relative` (или окна).
  - `fixed` — закреплён к окну браузера (например, шапка, которая всегда сверху).
  - `sticky` — ведёт себя как `relative`, пока не «прилипнет» к краю при скролле.
- **`top` / `right` / `bottom` / `left`**
  - Смещение элемента, если он `relative`, `absolute`, `fixed` или `sticky`.
- **`z-index`**
  - Очерёдность наложения элементов (кто «поверх» кого).
  - Работает только с элементами, у которых `position` не `static`.

### Цвета, фон и прозрачность

- **`color`**
  - Цвет текста.
- **`background-color`**
  - Цвет фона элемента.
- **`background-image`**
  - Картинка как фон: `background-image: url('image.png');`.
- **`background-repeat`**
  - Повторение фона: `no-repeat`, `repeat`, `repeat-x`, `repeat-y`.
- **`background-position`**
  - Позиция фона: `center`, `top left`, `50% 50%` и т.п.
- **`background-size`**
  - Размер фона: `cover` (заполнить с обрезкой), `contain` (уместить целиком), конкретные размеры.
- **`background`**
  - Сокращённая запись для всех фоновых свойств.
- **`opacity`**
  - Прозрачность от `0` (полностью прозрачный) до `1` (полностью непрозрачный).

### Текст и шрифты

- **`font-family`**
  - Какой шрифт использовать. Обычно указывают несколько: `"Inter", Arial, sans-serif`.
- **`font-size`**
  - Размер шрифта (`px`, `em`, `rem`, `%`).
- **`font-weight`**
  - Толщина: `normal`, `bold`, `100–900`.
- **`font-style`**
  - Стиль: `normal`, `italic`, `oblique`.
- **`line-height`**
  - Межстрочный интервал (высота строки). Можно задавать как число (`1.5`) или в единицах (`24px`).
- **`text-align`**
  - Выравнивание текста: `left`, `right`, `center`, `justify`.
- **`text-decoration`**
  - Подчёркивание и т.п.: `none`, `underline`, `line-through`.
- **`text-transform`**
  - Регистр: `uppercase`, `lowercase`, `capitalize`.
- **`letter-spacing`**
  - Расстояние между буквами.
- **`word-spacing`**
  - Расстояние между словами.

### Flexbox (раскладка элементов в строку/колонку)

Чтобы использовать Flexbox, у контейнера ставят `display: flex;`.

- **`flex-direction`**
  - Направление: `row` (по строке), `column` (по колонке).
- **`justify-content`**
  - Как распределять элементы по **главной оси**:
  - `flex-start`, `center`, `flex-end`, `space-between`, `space-around`, `space-evenly`.
- **`align-items`**
  - Выравнивание по **перпендикулярной оси**: `stretch`, `flex-start`, `center`, `flex-end`, `baseline`.
- **`flex-wrap`**
  - Разрешить перенос: `nowrap`, `wrap`, `wrap-reverse`.
- **`gap`**
  - Расстояние между элементами (работает и в Grid).
- **`flex`**
  - Краткая запись для `flex-grow`, `flex-shrink`, `flex-basis`, например: `flex: 1;`.
- **`align-self`**
  - Переопределяет `align-items` для одного конкретного элемента.

### CSS Grid (табличная сетка)

У контейнера: `display: grid;`.

- **`grid-template-columns`**
  - Описание колонок, например: `grid-template-columns: 1fr 1fr 1fr;` — три равные колонки.
- **`grid-template-rows`**
  - Аналогично, но для строк.
- **`gap`** / **`row-gap`** / **`column-gap`**
  - Отступы между ячейками сетки.
- **`justify-items`** / **`align-items`**
  - Выравнивание содержимого ячеек по горизонтали/вертикали.
- **`grid-column`**, **`grid-row`**
  - Задают, какие колонки/строки занимает конкретный элемент (например, `grid-column: 1 / 3;`).

### Скролл, переполнение и видимость

- **`overflow`**
  - Что делать, если содержимое не помещается:
  - `visible` — вылезает за границы.
  - `hidden` — лишнее обрезается.
  - `scroll` — всегда есть полосы прокрутки.
  - `auto` — полосы появляются только при необходимости.
- **`overflow-x` / `overflow-y`**
  - То же самое, но отдельно по горизонтали/вертикали.
- **`visibility`**
  - `visible` — видно, `hidden` — скрыто, но место остаётся.

### Тени, скругления, анимации

- **`border-radius`**
  - Скругление углов (например, `border-radius: 8px;`).
- **`box-shadow`**
  - Тень вокруг блока: `box-shadow: 0 4px 10px rgba(0,0,0,0.2);`.
- **`text-shadow`**
  - Тень у текста.
- **`transition`**
  - Плавный переход между состояниями:
  - `transition: all 0.3s ease;` — анимация всех изменяющихся свойств за 0.3 секунды.
- **`transform`**
  - Преобразования: `translate`, `scale`, `rotate`, `skew`.
  - Пример: `transform: scale(1.1);` — немного увеличить элемент.

---

### Прочее, часто встречающееся

- **`cursor`**
  - Вид курсора при наведении: `pointer`, `default`, `text`, `move` и др.
- **`outline`**
  - Внешний контур (часто используется для состояний фокуса у кнопок/инпутов).
- **`overflow-wrap` / `word-break`**
  - Управление переносами длинных слов/строк.
- **`object-fit`**
  - Как картинка вписывается в контейнер: `cover`, `contain` и др.
