# article_2

Этот репозиторий — **подробное дополнение к статье**: здесь собраны примеры кода, справочники и готовый сайт, чтобы можно было разобрать тему по шагам и потрользоваться материалами в работе.

---

## Что здесь есть

### [css](css/) — стили

Справочник по основным свойствам CSS: что такое `padding`, `margin`, `width`, `display`, `flex`, `grid` и т.д. В [css/README.md](css/README.md) кратко расписано, за что отвечает каждое свойство и как его применять.

---

### [js](js/) — варианты исполнения JavaScript в вебе

Набор простых примеров: счётчик нажатий, показ/скрытие блока, смена темы, форма с проверкой, калькулятор, секундомер и другие. Каждый пример — отдельная страница и свой `.js` файл. Удобно смотреть по одному и понимать, как подключать и использовать JS на странице.

**Оглавление примеров:** [js/index.html](js/index.html) (открой в браузере — там ссылки на все демо).

---

### [standart site](standart%20site/) — готовый сайт

Минимальный готовый сайт (HTML + CSS + JS): структура страницы, стили и простой скрипт. Можно открыть локально и использовать как шаблон или базу для своих экспериментов.
